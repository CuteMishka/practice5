### функции которые использовал:

* **re.findall()**

### использованный синтаксис:
* **Метасимволы**: `^` (начало), `$` (конец).
* **Последовательности**: `\d` (цифры), `\w` (буквы).
* **Квантификаторы**: `+` (один и более), `{12}` (ровно 12 для БИН).
